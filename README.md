# About-Us
This Project is part of final project of Computer Science degree in Ariel University. 
In this project we attend to connect the Moodle Forum to gmail address so the user will be able to connect the Moodle forum directly from his presonal address by writing mail to the specific address. In this project we work with : SQL, PHP and Gmail. At workspace: MySQL , Apache×ª Xampp, Moodle and IntelliJ .
